# 📚 NexoScienceSearch Android

**Versión Android 100% funcional** del buscador de artículos científicos.

Hecha con **Python + Kivy** para que puedas generar un APK real.

## ✨ Características

- Búsqueda en tiempo real
- Lista de artículos científicos (datos de ejemplo)
- Filtro por tema
- Interfaz moderna y fácil de usar
- Totalmente funcional

## 🚀 Cómo crear el APK

1. Instala buildozer:
   ```bash
   pip install buildozer cython
   ```
2. Ejecuta:
   ```bash
   buildozer android debug
   ```
3. El APK estará en la carpeta `bin/`

## 💻 Probar en PC

```bash
pip install kivy
python main.py
```
